# Vacuum-Fluorescent-Display-Driver
the cubeIDE files you'd need for driving a VFD. u8g2 only for startup animation-if you choose to omit it, then u8g2 is unnecessary.

## Notes
Looking at the display driver board facing you, the right side pins are as follows
From top to bottom
- Vin (5-12V)
- GND
- RX
- TX (unused)

Header info and UART baud rate can be found in the .ino
